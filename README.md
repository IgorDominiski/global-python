## Descrição do Projeto

Este projeto consiste no desenvolvimento de uma aplicação em Python voltada para o tema “Futuro do Trabalho”.
O objetivo do sistema é permitir que o usuário gerencie suas habilidades profissionais, registre níveis de competência e analise seu desempenho por meio de uma tabela de rendimento e gráficos gerados automaticamente.

A solução simula uma ferramenta essencial para o futuro do trabalho, onde a atualização contínua de competências e o uso de dados para avaliação de performance são fundamentais.

---

## Funcionalidades Principais

1. Cadastro de habilidades com nível de proficiência.
2. Listagem completa das habilidades registradas.
3. Criação de uma tabela de rendimento personalizada pelo usuário.
4. Geração de gráfico utilizando a biblioteca Matplotlib.
5. Menu interativo com validação de entrada e tratamento de exceções.
6. Uso de dicionários como base de dados interna.
